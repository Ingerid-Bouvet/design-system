## Figma

:warning: Rough notes about adding elevation in Figma. 

* Select the element you want to apply elevation to.  
* In the sidebar under **Effects**, pick the **Styles** menu. This is the button with the four dots.
  * :warning: Image example like: https://i.imgur.com/jH1JSog.png
* The **Styles** menu will open and you can choose the elevation you need.
  * :warning: Image example like: https://i.imgur.com/eYPY1Up.png
