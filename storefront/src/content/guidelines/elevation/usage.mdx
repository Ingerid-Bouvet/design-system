## Figma

### How to add
Assuming you have an element to add elevation style to---select the element, then "Effects" and open the style library (:warning: link to Getting Started > Figma, Finding components) and choose style under the "Elevation" section.


### How to use
Please see the Elevation overview section for more information on available elevation styles and when to apply which style.
