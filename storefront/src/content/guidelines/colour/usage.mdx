## Figma

### How to use
Select the layer you want to change the colour of, then locate the "Fill" section and open the style library (:warning: link to Getting Started > Figma, Finding components). The library contains many styles in different sections, in order of relevance.


### How to use
Please see the overview for more information on available styles and when to apply which style.
